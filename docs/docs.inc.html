Border Radius Classes

Class Description
r All corners (0.25rem)
round Fully rounded (9999rem)
rtl Top-left corner only
rtr Top-right corner only
rbr Bottom-right corner only
rbl Bottom-left corner only
rt Top corners (left + right)
rb Bottom corners (left + right)
rl Left corners (top + bottom)
rr Right corners (top + bottom)

Note: Add 0 suffix (e.g., r0, rt0) to cancel border radius.

Drop Shadow

Add the drop-shadow class to add a box shadow effect to any element. The shadow automatically adjusts for light and dark themes.

<div class="drop-shadow">Drop Shadow</div>
<div class="card drop-shadow">Card with Drop Shadow</div>
Drop Shadow
Card with Drop Shadow

Drop Shadow CSS Variables

Variable Description
--drop_shadow The active drop shadow (automatically set based on theme)
--drop_shadow__light Drop shadow for light theme
--drop_shadow__dark Drop shadow for dark theme (stronger for better visibility)

Disable Body Scrolling

Add the no-scroll class to the body to temparily disable body scrolling, this is used for things like dialogs and side menus that temporarily take over the page.

<button onclick="document.body.classList.toggle('no-scroll')">Toggle Body Scrolling</button>
