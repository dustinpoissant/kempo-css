Quick Start

Install via NPM:

npm install kempo-css

Or use the CDN:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/kempo-css@2/dist/kempo.min.css" />

Features

  • Lightweight - Small footprint, no JavaScript required
  • Utility-First - Composable utility classes
  • Responsive Grid - 12-column grid with breakpoints
  • Dark Mode - Built-in light/dark theme support
  • Elevation System - Z-index, shadows, and backgrounds
  • Customizable - CSS variables for easy theming

Includes

  • Typography - Headings, text, lists
  • Layout - Flexbox, grid, display utilities
  • Spacing - Padding, margins, negative margins
  • Forms - Inputs, labels, checkboxes, radios
  • Buttons - Colors, sizes, groups
  • Tables - Basic and responsive tables
